# serieavila

**Projeto para análise da Série Ávila**

## Ambiente 
Projeto gerado com **Ionic CLI v3.3.0**. Para atualizar a versão local:
> $ npm install -g ionic cordova

Pastas ```node_modules``` e ```./www``` estão especificadas no ```.gitignore```. Executar:
> $ npm install  <br>
> $ ionic serve
